---
title: Dependencies
description: Dependencies for NextCoinFlip
---

<Callout type="warning">**Warning** Install Vault and any Hologram plugin before installing NextCoinFlip. 
If not then NextCoinFlip will not work.</Callout>

| Plugin | Description | Required |
|--------|-------------|----------|
| Vault | Use for Essentials Economy | No |
| Essentials | Use to provide economy currencies for the plugin | No |
| PlaceholderAPI | Use to create a placeholders that can be used externally, more info at Placeholders | Yes |
